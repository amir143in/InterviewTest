1. Install Python: https://www.python.org/downloads/
2. Install pytest: pip install pytest
3. Install requests library: pip install requests
4. Install pycharm community version
5. Sign up for a free account on the Open Exchange Rates website: https://openexchangerates.org/signup
After signing up, you'll get an API key and the same use in the script "access_key".
6. Run the script within Pycharm
